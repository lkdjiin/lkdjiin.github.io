Première étape avant de commencer la construction de l'ordinateur 8 bits :
faire une horloge. Même si dans la version finale on utilisera un crystal
de 1MHz, pouvoir disposer d'une horloge réglable devrait être bien utile pour la
mise au point et le débuggage.

<!-- more -->

Je suis la vidéo [Astable 555 timer - 8-bit computer clock - part 1](https://www.youtube.com/watch?v=kRlSFm519Bo).
Un LM555, un condensateur, trois
résistances et hop, on a une horloge basique qui fait clignoter une LED.
Le plus long a été de remettre la main sur un adaptateur pour la _breadboard_.

{% img center /images/horloge001.gif %}

Le 555 est un composant avec lequel on peut faire une tonne de choses. Si
vous voulez voir de quoi il est capable regardez donc [par ici](https://www.talkingelectronics.com/projects/50%20-%20555%20Circuits/50%20-%20555%20Circuits.html).

J'ai fait un petit schéma sur Kicad pour montrer la valeur des composants que
j'ai utilisé. (_et je dois dire que je suis bien rouillé pour ce qui est
d'utiliser Kicad :D_)
