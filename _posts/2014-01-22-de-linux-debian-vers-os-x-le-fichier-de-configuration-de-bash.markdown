<!-- more -->

Où mettre son fichier de configuration pour Bash ?
----------------------------------------------------

Sous Linux, le fichier de configuration de Bash est généralement
placé dans `~/.bashrc`.

Sous OS X, il faut le mettre dans `~/.bash_profile`.
