Pour supprimer un fichier au clavier
----------------------------------------------------

Sur Linux, il faut appuyer sur la touche «Suppression» pour supprimer un
fichier depuis l'exporateur.

Sur OS X, il faudra appuyer sur «Meta Backspace».
